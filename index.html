Amazon Review Predictor

Platform

Website

Language

Python

Released

Winter 2017

Updated

Spring 2017

Summary

  This was a finished school project that I was lucky enough to work on. It gave me a practical outlet to experiment with machine learning.

  The algorithm takes components of a standard Amazon review, i.e. user name, review text, date posted, and then predicts what category the review belongs to, e.g. Pets, Books, or even Music. Using an ensemble of algorithms we were able to achieve an ~85% success rate on 57,000 reviews. The ensemble was comprised of 5 algorithms: Multinomial Multinomial Naïve Bayes, Bernoulli Naïve Bayes, Logistic Regression, Random Forests, and Gradient Boosting.
